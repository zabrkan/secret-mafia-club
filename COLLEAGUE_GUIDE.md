# 🔪 How to help with the Secret Mafia Club website

Hey! This is our Mafia game tracker: **https://secret-mafia-club.netlify.app**

You can help change it just by **describing what you want in plain English** — an AI assistant
called Claude does all the actual work. You never have to read or write any code.

---

## ⚡ First, the easy shortcut
If you only want to **add game results** or **fix a player's name**, you don't need any of the steps
below — that lives in our **Google Sheet**. Just edit the sheet and the website updates the next time
you refresh it. (Ask «YOUR NAME» for the sheet link.)

Everything below is only for changing the **website itself** — its look, new features, new stats.

---

## What you'll need (just two accounts)
1. **A free GitHub account** — sign up at **github.com**. Then send your GitHub **username** to
   «YOUR NAME» so they can give you access. You'll get an email invitation — open it and click **Accept**.
2. **A Claude account that has Claude Code** — go to **claude.ai/code** and sign in. (If it lets you
   start/open a project, you're all set. If not, you may need to upgrade your Claude plan.)

---

## One-time setup (about 5 minutes)
1. Accept the GitHub email invite from «YOUR NAME».
2. Open **claude.ai/code**.
3. When it asks, **connect your GitHub account**.
4. Pick the project called **secret-mafia-club** from the list.

Done — Claude can now see the website.

---

## How to make a change (every time)
1. Open **claude.ai/code** and choose the **secret-mafia-club** project.
2. **Type what you want**, like you're texting a coworker. For example:
   - "Change the homepage title from BATTLE RECORDS to SEASON 2."
   - "Add a new leaderboard stat showing each player's longest losing streak."
   - "Make the Mafia red a little brighter."
   - "Put a trophy emoji next to the #1 player."
3. Claude reads the project, makes the change, and **saves it as a 'pull request'** — think of it as a
   suggested change waiting for a thumbs-up.
4. «YOUR NAME» gets notified, gives it a quick approval, and **the website updates automatically.** 🎉

You don't need to understand any code. If Claude asks you a question, just answer in plain English.
If something looks wrong, tell Claude *"that's not right, please undo it"* — or message «YOUR NAME».

---

## A few tips
- Be specific: *"make the title green"* works better than *"make it look nicer."*
- One change at a time is easiest to follow.
- There's a file in the project called **CLAUDE.md** — Claude reads it on its own, so it already knows
  how our site works. You never need to open it.
- Stuck on anything? Just message «YOUR NAME».
